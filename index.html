Chrome DevTools
is a set of tools for web developers
built directly into the browser

Open DevTools

  1. Press Command + Option + C (Mac) or Control + Shift + C (Windows, Linux)
  2. Press Command + Option + J (Mac) or Control + Shift + J (Windows, Linux)
  3. Press Command + Option + I (Mac) or Control + Shift + I (Windows, Linux)
  4. Call the context menu on the inspected page and select the "Inspect" item

Elements panel

  • View And Change The DOM
  • View And Change CSS
  • Edit Styles
  • Edit the DOM...

Console panel

  • View the messages sent from the console.log
  • Run JavaScript code to interact with the web page...

Source panel

  • Debug JavaScript
  • Stop Code With Breakpoints
  • Run Snippets Of Code From Any Page
  • Find Unused JavaScript...

Network panel

  • Log network activity
  • Simulate a slower network connection
  • Inspect a resource's details
  • Search network headers and responses...

Performance panel

  • Optimize Website Speed
  • Analyze Runtime Performance...

Memory panel

  • Fix Memory Problems
  • JavaScript CPU Profiler...

Application panel

  • Debug Progressive Web Apps
  • Inspect and Manage Storage, Databases, and Caches
  • Inspect and Delete Cookies
  • Inspect Resources...

Security panel

  • Non-secure main origins
  • Mixed content
  • View connection and certificate information...

Lighthouse panel

  • Audit web pages
  • Page quality report...
